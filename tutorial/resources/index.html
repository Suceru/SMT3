    安卓资源¶

    apk修改、文件管理¶

    MT管理器

    双栏式文件管理器，可批量删除、添加安装包内文件，修改后需手动签名

    APKCrack

    功能纯粹的apk（安卓软件安装包）修改软件，操作简单

    ES文件浏览器

    文件管理器的集大成者，能够创建FTP服务器，与其他设备快速传输文件

    Content.pak处理¶

    SCPaker

    第三代Content.pak解包打包工具，由修改是种乐趣基于第一代工具SCPacker开发

    你知道吗？

    Content.pak解包打包工具第一代SCPacker和第二代scpak都是由百度贴吧电离的Tesla开发，除此之外，他还开发有SC with JS

    源代码相关¶

    dnSpy

    非常方便的dll、exe修改工具，可编辑源码

    NETFX Tools

    内有ildasm.exe，可反编译exe、dll文件为il，亦有和ILSpy相同的以不同语言查看的功能
    此处提供的版本提取自C:\Program Files (x86)\Microsoft SDKs\Windows\v10.0A\bin\NETFX 4.6 Tools

    IL Assembler

    简称ilasm，可将IL语言编写的文件转换为exe、dll文件，使用控制台执行
    此处提供的版本提取自C:\Windows\Microsoft.NET\Framework64\v4.0.30319

    Visual Studio

    开发新dll首选IDE，安装时只需勾选Windows - .NET 桌面开发
